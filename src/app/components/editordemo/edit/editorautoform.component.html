Edit current user
Techniques used

This form makes use of the <ac-autoform> component. An object with several properties, decorated with UI instructions, created a whole form. 

 <form (ngSubmit)="saveUser()" [formGroup]="myForm" role="form" class="row">

    <fieldset>

      <legend>Edit current user</legend>

      <ac-autoform [userForm]="myForm"></ac-editor>
>

The form needs a name, here we used 'myForm'. The component AcAutoForm reads the associated view model and creates all the fields including validation, label, and data type specific information.

The decorators can be used to achieve the following features:

  • Hide fields (exclude) from being shown (for IDs, primary keys etc)
  • Accept order instructions to show in particular order
  • Group fields into fieldsets and order the sets as well as the fields within a group
  • Change all labels
  • Control all kind of validations, including the error messages shown

The form tag itself, buttons and other form related stuff is not part of the control. The values are two-way bound to a model, that will change automatically.
