SVOGV is an ongoing project that helps Angular developers to create awesome apps quite fast.

Purpose

This is a project that helps creating sophisticated Angular components. It's not really just for adding to packages and using, but more for learning. The will be a book available that explains the making of completely.

The major part is what most projects really need: a grid component. This grid can:

  • Show any data, creating headers and rows dynamically
  • Format all kind of data flexibly
  • Can add and remove columns flexibly
  • Has paging
  • Has sorting
  • Has filtering on the client side
  • Can edit data inline

Just check all the features out in the Features section at the left.
