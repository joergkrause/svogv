A simple demo for the grid showing the basic features. Almost completely controlled by a distinct view model through decorators.
Grid Features

The grid below is using this code:

<ac-datagrid [model]="model"
             [showActions]="false"
             [columnStyle]="{{ '{' }} footer: {{ '{' }} 'background-color': 'silver' },
                              first: {{ '{' }} 'background-color': 'orange' },
                              header: {{ '{' }} color: 'white', 'background-color': 'black' },
                              headerSortButton: {{ '{' }} color: 'yellow' }
                             }" >
</ac-datagrid>

It shows how to change the styles of the particular parts of the grid.
